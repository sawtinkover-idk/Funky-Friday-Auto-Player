an ahk script that plays funky friday on roblox without input

all you have to do is run one or two times to find the right height of where the hold lines exist right above the middle then where all of the different button locations are i might have a coordinate finder in here if i do there will be a not in here on how to use it then you just copy the tail color and the block color (make sure the block and tail arent the same color as well as dont put the colors to white) to the script set the inputs to however you want them and i suggest playing with no lobby active since this script will add lag to the game.
